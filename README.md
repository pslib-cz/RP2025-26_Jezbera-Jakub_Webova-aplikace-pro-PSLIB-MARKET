#  PSLIB Market

PSLIB Market je moderní webová aplikace určená pro studenty, která zprostředkovává jednoduchý a bezpečný nákup a prodej použitých učebnic 

Cílem projektu je vytvořit centralizované místo, kde mohou studenti snadno najít potřebné materiály nebo naopak prodat ty, které už nevyužijí, a to s důrazem na bezpečnost prostřednictvím školního přihlašování.

## 🛠️ Technologie
* **Frontend:** React, TypeScript, Vite
* **Backend:** C# ASP.NET Core Web API, Entity Framework Core
* **Autentizace:** OIDC (OpenID Connect) přes školní server (SSO)
